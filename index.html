Tny Bot

A simple, customizable bot for discord.

!roleshelp Prints the help for these commands !roleshelp, !biashelp None
!setmainrole Sets a main role, or list of main roles.
Only one of these can be used by a member at a time. 		!setmainrole Robot, Zombie Manage Roles
!setrole Sets a role, or list of roles.
Unlike main roles, members can have multiple of them 		!setrole Robot, Zombie Manage Roles
!delrole Deletes a self assign role !delrole Zombie Manage Roles
