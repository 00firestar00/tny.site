Tny Bot

A simple, customizable bot for discord.

Commands

Utility Roles Notifications
Command Permissions Description Examples / Aliases
!help None Shows the help menu !help, !help Roles
!invite None Gets the invite link for Tny !invite
!hello None Tells Tny to say hello (random phrase) !hello
!joined None Prints when a user joined the server !joined @tnybot
!clear Manage Messages Clears chat messages !clear, !clear 100
!say None Tells Tny to say something !say I'm the best bot ever!
!addemoji Manage Emojis Add an emoji to the server using a url !addemoji http://tny.site/images/tnybot.png
!emoji None Lists the custom emojis on this server !emoji, !emojis
!listserveroles None Lists the roles on the server. None
Command Permissions Description Examples / Aliases
!roleshelp None
Manage Roles (for admin help)		 Prints the help for these commands !roleshelp
!roleshelp admin
!setmainrole Manage Roles Sets a main role, or list of main roles.
Only one of these can be used by a member at a time. 		!setmainrole Robot, Zombie
!setrole Manage Roles Sets a role, or list of roles.
Unlike main roles, members can have multiple of them 		!setrole Robot, Zombie
!delrole Manage Roles Deletes a self assign role !delrole Zombie
!listmainrole None Lists the main roles on the server (can only have 1 main role) !listmainrole
!listrole None Lists the roles on this server !listrole
!mainrole None Sets a main role !mainrole Robot
!role None Sets a role !role Zombie
!clearrole None Clears your roles !clearrole
Command Permissions Description Examples / Aliases
!notify None Adds a new notification
You will get a PM when this word is mentioned
It also accepts regex statements 		!notify tny, !notify ti?ny(bot)?
!notify_pinned None Tny will notify you when someone pins a message !notify_pinned, !notifyPinned
!notifications None Tny will send you a PM with the list of notifications !notifications
!delnotify None Deletes a notification !delnotify tny
